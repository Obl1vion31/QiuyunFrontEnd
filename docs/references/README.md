# 参考资料

本目录用于保存项目参考资料。

参考资料是产品、设计和技术判断的输入，不等同于最终决策。经过确认的结论应写入对应的正式文档，例如 `docs/PRODUCT.md`、`docs/STRUCTURE.md`、`docs/DESIGN.md` 或 `docs/DECISIONS.md`。

## 可放入的内容

- 智能文档截图；
- PDF 参考文件；
- 竞品链接；
- 设计参考；
- 品牌素材；
- 内容素材；
- 技术资料；
- 原始访谈或讨论记录。

## 使用原则

- 原始资料尽量保留来源和日期。
- 不要在参考资料目录中直接写最终产品结论。
- 如果参考资料之间存在冲突，应在正式文档中指出冲突，而不是直接覆盖。
- 视觉参考可以保留原文件，设计文档只记录解读和采用原则。
