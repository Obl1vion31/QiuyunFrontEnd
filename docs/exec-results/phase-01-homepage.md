## 4. 技术实现结果

当前技术栈：

- Astro 4
- React 18
- TypeScript
- pnpm

实现方式：

- 首页是 Astro 静态页面。
- React Islands 能力已通过集成保留，但当前首页没有强制加入复杂交互。
- 首页内容集中维护在 `src/data/home.ts`。
- `docs/references/Hero_Ref.md` 只作为首页应包含内容的范围来源。
- 页面没有直接渲染 `Hero_Ref.md`，也没有复刻其 Markdown 格式。

版本说明：

- 当前本地 Node 版本为 `18.19.1`。
- Astro 5 对 Node 版本要求更高，因此当前固定使用 Astro 4 + React 18。
- 未来升级 Node 后，可再评估是否升级 Astro 主版本。

## 5. 本地启动和访问方式

安装依赖：

```bash
pnpm install
```

启动开发服务器：

```bash
pnpm dev
```

打开终端输出的本地地址。

上一次运行地址为：

```text
http://127.0.0.1:4322/
```

如果端口被占用，Astro 会自动切换到下一个可用端口，应以终端输出为准。
